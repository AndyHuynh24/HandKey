# üñêÔ∏è Handkey ‚Äì Gesture-Controlled Computer Interaction
#### Handkey is a machine-learning‚Äìpowered application that lets you control your computer using hand gestures and movement.
Trained on 12 distinct gesture classes, Handkey recognizes dynamic hand movements in real time and maps them to custom shortcuts, commands, actions and mouse movement ‚Äî giving you a completely touch-free control experience.

## üöÄ Features

- üéØ **12 Well-Trained Gestures** ‚Äì Each gesture is recognized with high confidence and minimal latency.  
- üí° **Custom Action Mapping** ‚Äì Bind any gesture to actions like:
  - Keyboard shortcuts  
  - Mouse control  
  - Opening files or folders  
  - Typing preset text  
- ‚öôÔ∏è **Real-Time Detection** ‚Äì Runs smoothly on standard hardware using OpenCV and TensorFlow.  
- üß† **Optimized ML Model** ‚Äì LSTM/GRU-based sequence classifier trained on thousands of gesture samples with rotation and lighting augmentation.  
- üíª **Cross-Platform Ready** ‚Äì Works on macOS, Windows, and Linux.  
- üß© **Modular Design** ‚Äì Easy to integrate with other automation tools.
